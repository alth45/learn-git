Auto Testing
Test otomatis
setiap push
on: push
  branches: [main]

jobs:
  test:
    steps:
      - npm install
      - npm test

Auto Deploy
Deploy otomatis
ke GitHub Pages
  deploy:
    needs: testing
    steps:
      - uses: actions-gh-pages
        with:
          publish_dir: ./
